# Laporan Presentasi Aplikasi Rental Mobil

## 1. Pembuka

Assalamu'alaikum warahmatullahi wabarakatuh.

Selamat pagi/siang, pada kesempatan ini saya akan mempresentasikan aplikasi Rental Mobil bernama **Timbang Mlaku Transport**. Aplikasi ini dibuat menggunakan Flutter untuk membantu proses penyewaan mobil menjadi lebih praktis, rapi, dan mudah digunakan oleh pelanggan maupun admin.

Tujuan utama aplikasi ini adalah menyediakan layanan rental mobil digital yang menampilkan katalog kendaraan, proses booking, perhitungan biaya, riwayat pesanan, serta pengelolaan data mobil dan user melalui halaman admin.

## 2. Problem Statement

Pada layanan rental mobil konvensional, proses pemesanan sering masih dilakukan secara manual melalui chat, telepon, atau pencatatan langsung. Cara ini dapat menimbulkan beberapa masalah, seperti:

- Pelanggan sulit melihat daftar mobil yang tersedia secara cepat.
- Informasi harga, tipe mobil, dan status ketersediaan tidak selalu tersusun rapi.
- Proses pemesanan membutuhkan komunikasi berulang.
- Perhitungan biaya sewa dapat memakan waktu dan berisiko salah hitung.
- Admin perlu mencatat data mobil, user, dan pesanan secara manual.

Masalah tersebut membuat proses rental mobil menjadi kurang efisien, baik untuk pelanggan maupun pengelola rental.

## 3. Solusi

Solusi yang ditawarkan adalah aplikasi Rental Mobil berbasis Flutter yang menyediakan fitur lengkap dalam satu aplikasi.

Fitur utama aplikasi:

- **Halaman beranda** untuk menampilkan identitas layanan, keunggulan, cara pemesanan, dan mobil populer.
- **Katalog mobil** untuk melihat daftar armada, gambar mobil, tipe, warna, tahun, harga per jam, dan status ketersediaan.
- **Akun user** untuk register atau login sebelum melakukan booking.
- **Form booking** untuk memilih mobil, tanggal pickup, jam pickup, tanggal dropoff, dan jam dropoff.
- **Invoice otomatis** yang menampilkan durasi sewa dan total biaya berdasarkan harga per jam.
- **Riwayat booking** untuk menyimpan pesanan yang sudah dibuat.
- **Dashboard admin** untuk mengelola data mobil, data user, data pesanan, dan melihat ringkasan pendapatan.

Dengan aplikasi ini, pelanggan dapat melakukan booking dengan alur yang lebih jelas, sedangkan admin dapat mengelola operasional rental secara lebih terstruktur.

## 4. Preview Demo Aplikasi

Pada bagian demo, alur aplikasi dapat ditunjukkan sebagai berikut:

1. **Beranda**
   - Menampilkan nama aplikasi **Timbang Mlaku Transport**.
   - Menampilkan tombol booking, keunggulan layanan, langkah pemesanan, dan mobil populer.

2. **Katalog Mobil**
   - User dapat melihat daftar mobil.
   - Setiap mobil menampilkan gambar, nama, plat nomor, tipe, warna, tahun, harga per jam, dan status tersedia atau habis.
   - Jika mobil tersedia, user dapat menekan tombol booking.

3. **Akun User**
   - Sebelum booking, user harus login atau membuat akun.
   - Data user berisi username, email, password, NIK, nama depan, dan nama belakang.

4. **Booking Mobil**
   - User memilih mobil yang tersedia.
   - User menentukan tanggal dan jam pickup serta dropoff.
   - Aplikasi menghitung durasi sewa dan total biaya secara otomatis.
   - Setelah konfirmasi, booking berhasil dibuat.

5. **Riwayat Booking**
   - Pesanan yang sudah dibuat tampil dalam daftar riwayat.
   - Informasi yang ditampilkan meliputi nama mobil, nama penyewa, NIK, jadwal pickup, jadwal dropoff, durasi, dan total harga.

6. **Dashboard Admin**
   - Admin dapat login ke halaman khusus.
   - Admin dapat menambah, mengedit, dan menghapus data mobil.
   - Admin dapat mengelola data user.
   - Admin dapat melihat jumlah mobil, mobil tersedia, jumlah user, jumlah pesanan, dan total pendapatan.

## 5. Kesimpulan

Aplikasi Rental Mobil **Timbang Mlaku Transport** membantu proses penyewaan mobil menjadi lebih mudah dan efisien. Aplikasi ini memudahkan pelanggan dalam melihat mobil, membuat akun, melakukan booking, dan melihat invoice. Di sisi admin, aplikasi membantu pengelolaan data mobil, user, pesanan, serta ringkasan pendapatan.

Dengan adanya aplikasi ini, proses rental mobil yang sebelumnya manual dapat menjadi lebih digital, cepat, dan terorganisir.
